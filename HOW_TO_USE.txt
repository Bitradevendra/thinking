# 🚀 HOW TO USE YOUR DEEP THINKING PROMPT

## ✅ STEP-BY-STEP INSTRUCTIONS:

### 1. Open the Web Interface
```
http://localhost:5000
```
(The server should still be running. If not, run: `python web_server.py`)

### 2. Copy the Prompt
Open this file: **`PROMPT_FOR_WEB_UI.txt`**
- Select ALL text (Ctrl+A)
- Copy it (Ctrl+C)

### 3. Paste into Web UI
- In the "Describe your problem:" text box
- Paste the entire prompt (Ctrl+V)

### 4. Upload Your Trading Files
- Click "📎 Choose Files" or drag-and-drop
- Upload ALL your trading data files:
  - UT Bot.txt
  - ACC_F1.txt
  - ADANIENT_F1.txt
  - ADANIPORTS_F1.txt
  - ALUMINIUM_F1.txt
  - ALUMINIUM_F2.txt
  - (and all other trading data files)

### 5. Set Deep Thinking Parameters
- **Iterations**: `1800` (for 30 minutes of deep thinking)
- Advanced settings (optional):
  - Max Depth: `8`
  - Branches per Node: `5`
  - Min Score: `0.2`

### 6. Start Deep Reasoning
- Click the **"🚀 Start Deep Reasoning"** button
- The system will now think for ~30 minutes!

### 7. Watch the Progress
You'll see:
- Progress bar moving
- Iterations: X / 1800
- Nodes Explored: growing number
- Elapsed Time: counting up
- Status messages changing

### 8. Wait for Results
After ~30 minutes, you'll get:
- Complete pattern analysis
- Mathematical formulas
- Pattern linkages
- Trading recommendations
- Success rate predictions
